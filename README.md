## Info

This is a example repo for solving https://tomdalling.com/toms-data-onion/

More details on [puzzle.txt](puzzle.txt)

## Usage

```shell script
$ pip3 install virtualenv
$ virtualenv venv
$ source venv/bin/activate
$ pip3 install -r requirements.txt
$ python3 src/main.py
```
